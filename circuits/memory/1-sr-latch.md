# SR 锁存器

## 存在意义

组合逻辑电路的输出只由当前输入决定，**没有记忆能力**。为了存储信息，我们需要一种能"记住"状态的电路。

**SR 锁存器（Set-Reset Latch）** 是最基本的 **记忆单元**，可以存储 **1 位二进制数据**（0 或 1）。它是构成寄存器、内存等所有存储电路的基础。

## 原理

SR 锁存器有两个输入和两个输出：

| 信号 | 名称 | 说明 |
|------|------|------|
| S    | Set（置位） | 将输出 Q 设为 1 |
| R    | Reset（复位） | 将输出 Q 设为 0 |
| Q    | 输出 | 存储的状态值 |
| Q̅   | 反相输出 | Q 的逻辑非（互补输出） |

### 电路结构

SR 锁存器由 **两个交叉耦合的或非门（NOR）** 组成：

- 上面 NOR 门的输出是 Q，其输入为 R 和下面门的输出 Q̅
- 下面 NOR 门的输出是 Q̅，其输入为 S 和上面门的输出 Q

### 工作逻辑

- **S = 0, R = 0** → **保持**：输出 Q 维持不变（锁存状态）
- **S = 1, R = 0** → **置位**：Q 被设为 1
- **S = 0, R = 1** → **复位**：Q 被设为 0
- **S = 1, R = 1** → **禁止**：Q = Q̅ = 0，破坏互补关系，应避免

## 真值表

| S | R | Q（新） | Q̅（新） | 状态 |
|:-:|:-:|:------:|:-------:|:----:|
| 0 | 0 |  Q 保持不变 |  Q̅ 保持不变 | **保持** |
| 1 | 0 |   1    |    0    | **置位** |
| 0 | 1 |   0    |    1    | **复位** |
| 1 | 1 |   0    |    0    | **禁止** |

> 也可以用 **与非门（NAND）** 实现 SR 锁存器，此时 S 和 R 为低电平有效。

## 说明

- SR 锁存器是 **时序逻辑电路** 的起点——输出不仅取决于当前输入，还取决于之前的状态（即"记忆"）。
- 通过组合多个锁存器可以构成 **寄存器**，用于存储多位数据。
- SR 锁存器存在 **禁止状态**（S=R=1），且无法同时置位和复位，因此在实际中常使用改进的 **D 锁存器** 或 **JK 触发器**。
