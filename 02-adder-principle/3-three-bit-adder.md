# 构建三位加法器

全加器搞定之后，拼在一起就能算多位二进制数了。这里用三个全加器级联做一个 **3 位加法器**。

## 接线说明

三个全加器的 Cout → 下一级的 Cin 依次连接。

**输入**（两排杜邦线，从左到右为高位到低位）：
- 第一排：`A2 A1 A0`（被加数）
- 第二排：`B2 B1 B0`（加数）

**输出**（4 个 LED，从左到右为高位到低位）：
- `C3 C2 C1 C0`（最高位 C3 是进位输出）

## 实测

### 6 + 7 = 13（二进制 110 + 111 = 1101）

```
   A2 A1 A0 =   1  1  0
+  B2 B1 B0 =   1  1  1
─────────────────
= C3 C2 C1 C0 = 1  1  0  1
```

![三位加法器 6+7](../images/02-adder-principle/02-3-adder-6+7.jpg)

### 3 + 5 = 8（二进制 011 + 101 = 1000）

```
   A2 A1 A0 =   0  1  1
+  B2 B1 B0 =   1  0  1
─────────────────
= C3 C2 C1 C0 = 1  0  0  0
```

![三位加法器 3+5](../images/02-adder-principle/02-3-adder-3+5.jpg)
