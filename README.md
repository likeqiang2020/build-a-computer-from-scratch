## 序言

### 为什么要学"造计算机"？

现代计算机是层层抽象叠加的产物：

```
高级语言 → 汇编 → 机器码 → 指令集架构(ISA) → CPU → 逻辑门 → 晶体管
```

大多数人停留在最上层。但当你**亲手从逻辑门搭出一颗 CPU**，你会真正理解：

- 程序到底是什么
- 内存为什么能"记住"数据
- 加法、跳转、寻址在硬件层面如何发生
- 编译器、操作系统究竟在做什么

### 笔记风格

每条笔记遵循统一的模板：

| 要素 | 说明 |
|------|------|
| **概述** | 该组件是什么，解决什么问题 |
| **原理** | 输入/输出定义，逻辑表达式，电路结构 |

| **图示** | 电路图或逻辑示意图 |
| **说明** | 关键洞察、易错点、与后续章节的关联 |

## 🏗️ 核心概念速查

### 逻辑门符号速查

| 门 | 布尔表达式 | 逻辑符号 | 说明 |
|:--:|:----------|:--------:|:----|
| NOT | $\overline{A}$ | !A | 取反 |
| AND | $A \land B$ | A·B | 全 1 出 1 |
| OR | $A \lor B$ | A+B | 有 1 出 1 |
| NAND | $\overline{A \land B}$ | !(A·B) | 全 1 出 0 |
| NOR | $\overline{A \lor B}$ | !(A+B) | 全 0 出 1 |
| XOR | $A \oplus B$ | A⊕B | 不同出 1 |

> 💡 **技巧**：NAND 和 NOR 是**通用门**——只用 NAND（或只用 NOR）就能搭出所有其他门。

### 从晶体管到 CPU 的抽象层级

```
晶体管 → 逻辑门 → 组合电路 → 时序电路 → 运算单元 → CPU → 计算机
```

每一层都封装了底层的复杂性，向上层提供更强大的"积木块"。
